# Next.js MDX

MDX is a compatibility layer provided by Next.js and other packages to transform MDX content into HTML syntax.
This reduces the time required to create long content, allowing users to focus solely on writing text. MDX, a
lightweight markup language, is particularly useful for blogs as it enables users to create text without worrying
about structural content or components.

## Installation

{/* prettier-ignore */}
<Code lang="bash">
npm install @next/mdx @mdx-js/loader @mdx-js/react @types/mdx
\# or
pnpm add @next/mdx @mdx-js/loader @mdx-js/react @types/mdx
</Code>
